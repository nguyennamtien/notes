h2. Reference

h3. Adding anchor links

Set your anchor links in your header tags by adding @(#yourAnchorId)@ like so:

<pre>
h1(#yourAnchorId). 
</pre>

Link to it by using the the normal link <code>"Text":#yourAnchorId</code>

h3. Adding non textile html blocks

Just add @<notextile>@ blocks like so
	
<pre>
	<notextile>
	<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
	<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="en">
	  <head>
	    <meta http-equiv="Content-Type" content="text/html; charset=utf-8"/>
	    <title>Title</title>
	  </head>
	  <body>
	</notextile>
</pre>

Especially useful for including header information.

h2. FAQ/Problems

h3. Commands aren't replaced leaving @<command>. content@ in output

This can happen if there is a tab after the command instead of the space, which is not distuingishable from one another if a tab represents four spaces and the command is two spaces long.

Remove the tab and replace it with a space character.
