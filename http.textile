h3. POST a file

Given this form

bc. <form method="POST" enctype='multipart/form-data' action="upload.cgi">
	<input type=file name=upload>
	<input type=submit name=press value="OK">
</form>

you can POST it with

bc. curl -F upload=@localfilename -F press=OK [URL]
